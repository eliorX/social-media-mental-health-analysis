# Social Media & Mental Health in Adolescents

## Project Overview
This project analyzes behavioral and lifestyle data from 1,000+ students aged 13–19 
to identify which factors most strongly predict depression risk. 
The goal is to provide actionable insights for schools, counselors, and parents.

## Key Findings
- Sleep is the strongest protective factor against depression
- High social media usage (6+ hours/day) correlates with increased depression risk
- Stress level is a stronger predictor of depression than screen time addiction alone
- Students aged 15 and 19 show the highest vulnerability
- Academic GPA has near-zero correlation with depression

## Tools Used
- Google Sheets (data cleaning, pivot analysis, dashboard)
- Excel (advanced pivot tables, correlation analysis)

## Dataset
- 1,000+ student records
- Variables include: age, gender, daily social media hours, platform used, 
  sleep hours, stress level, anxiety level, addiction level, GPA, depression label
